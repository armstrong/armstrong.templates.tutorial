armstrong.templates.tutorial
============================
Provides a basic Armstrong project for use with the `Armstrong tutorial`_.


Usage
-----
Please see the `Armstrong tutorial`_ for how to use this template.

.. _Armstrong tutorial: http://docs.armstrongcms.org/getting-started/5-minutes/


Installation & Configuration
----------------------------
You can install the latest release of ``armstrong.templates.tutorial`` using `pip`_:

::

    pip install armstrong.templates.tutorial

No configuration is required for this component.

.. _pip: http://www.pip-installer.org/

State of Project
----------------
Armstrong is an open-source news platform that is freely available to any
organization.  It is the result of a collaboration between the `Texas Tribune`_
and `Bay Citizen`_, and a grant from the `John S. and James L. Knight
Foundation`_.

To follow development, be sure to join the `Google Group`_.

``armstrong.templates.standard`` is part of the `Armstrong`_ project.  You're
probably looking for that.
